# 🚀 Productivity Dashboard

A modern **all-in-one productivity dashboard** built using **HTML, CSS, and Vanilla JavaScript**.  
This project helps users manage daily tasks, plan their day, stay motivated, track goals, and focus using a Pomodoro timer — all in one place.

---

## ✨ Features

### 📝 To-Do List
- Add tasks with optional details
- Mark tasks as important
- Mark tasks as completed
- Persistent storage using **LocalStorage**

### 📅 Daily Planner
- Hour-wise planning from morning to night
- Auto-saves input using **LocalStorage**
- Clean and minimal UI for daily scheduling

### 💡 Motivation Section
- Displays a random **Quote of the Day**
- Quotes fetched using an external API

### ⏱️ Pomodoro Timer
- 25-minute work session
- 5-minute break session
- Start, pause, and reset functionality
- Visual session indicator (Work / Break)

### 🎯 Daily Goals
- Add daily goals
- Mark goals as completed
- Live progress tracking with percentage indicator

### 🌦️ Weather & Time
- Real-time weather data (temperature, condition, humidity, wind)
- Live date and time display
- City-based weather fetching using Weather API

### 🎨 Theme Switcher
- Multiple color themes
- Dynamically updates CSS variables
- Improves user personalization

---

## 🛠️ Tech Stack

- **HTML5** – Structure
- **CSS3** – Styling & Layout
- **JavaScript (ES6)** – Logic & Interactivity
- **LocalStorage API** – Data persistence
- **Weather API** – Real-time weather data
- **Quotable API** – Motivational quotes
